Project highlights

(2023) TORAX is an open-source, JAX-based simulator for tokamak core transport, i.e. a simulator for how the physics within plasma inside a nuclear reactor evolves over time.

(2023) RoboCat is a self-improving robotic agent that learns to solve tasks in the real world and improves its performance over time. A single model was able to solve tasks on different tasks and different robots.

(2021) dm_robotics is an open-source robotics library that provides tools to build environments and scripted agents for use in simulation and real-world robotics.

(2021) RGB Stacking is a real and simulation robotics benchmark for vision-based object manipulation, where we trained several RL-based agents. Code for the assets and environments is available on GitHub.
